# Milestone - Corrigir link de Categoria com Especificação tecnica do Produto

## Descrição
Hoje as especificações técnicas do produto foram desenhadas de uma forma que elas podem ser diferentes para cada produto, e isso cria um problema no momento de compararmos os produtos ou matermos um padrão dos dados das especificações técnicas, pois cada produto tem uma estrutura diferente. O objetivo dessa tarefa é criar um link entre a categoria do produto e as especificações técnicas, para que possamos ter um padrão e facilitar a comparação entre os produtos.
A ideia é que ao criar um produto, as categorias tragam as especificações técnicas associadas a essa categoria, e o usuário possa preencher essas especificações técnicas de acordo com o produto que está criando. Dessa forma, teremos um padrão nos nomes dos atributos das especificações técnicas e poderemos comparar os produtos de forma mais eficiente e ter um padrão nos dados das especificações técnicas.

## Tarefas
- [ ] Criar um link entre a categoria do produto e as especificações técnicas
- [ ] Ao criar um produto, as categorias devem trazer as especificações técnicas associadas a essa categoria
- [ ] Trazer um campo de autocomplete para os nomes de atributos das especificações técnicas, para que o usuário possa escolher entre os atributos já existentes ou criar um novo atributo caso necessário
