# Product Data Schema

All types import from `@/api/stetsom/model`. Never define API shapes inline — always import from the barrel.

## I18nString

Multilingual fields use `I18nString`: `{ pt: string, en?: string, es?: string }`.
- `pt` is always present (required). `en` and `es` are optional.
- Backend locale key is `pt` — **not** `pt-BR`. When picking text for display, use `pt` as the primary key.

## Core Types

**Product**
- `id`, `sku?`, `name: I18nString`, `slug: I18nString`, `description?: I18nString | null`
- `status: 'ACTIVE' | 'DISCONTINUED' | 'DRAFT'`
- `is_discontinued`, `is_featured`, `is_spotlight` (booleans)
- `available_locales: string[]`, `launch_date?: string` (ISO 8601)
- `category_id`, `line_id?`, `template_id?`
- `highlight_attributes: string[]`
- `images: ProductImage[]`, `variants: Variant[]`, `page_blocks: ProductBlock[]`, `files: ProductFile[]`
- `created_at`, `updated_at`, `created_by`

**Variant** (formerly `ProductVariation`): `{ variant_id, name, order, attributes: VariantAttr[] }`

**VariantAttr** (formerly `ProductSpec`): `{ attribute, value, order }`

**ProductBlock**: `{ block_id, type: 'IMAGE'|'VIDEO'|'HTML'|'MODEL3D'|'TEXT', order, data: ProductBlockData }`

**ProductFile**: `{ id, file_url, type: 'MANUAL'|'CATALOG'|'CERTIFICATE'|'IMAGE'|'OTHER', version, is_active, ... }`

**Category**: `{ id, name: I18nString, slug: I18nString, lines: Line[], order, created_at, updated_at }`

**Line** (formerly `Subcategory`): `{ line_id, name: I18nString, slug: I18nString, order }`

**ProductCardItem** (light catalog item): `{ id, slug, name, category, thumbnail_url, href, status, is_discontinued, highlighted_specs }`

## Key Rules

- `variants[].attributes[]` ordered by `order` ascending — never reintroduce flat `specifications`
- `page_blocks` ordered by `order` ascending — sort before rendering
- `files.version` increments per `product_id + type` — show highest active version only (`is_active: true`)
- `available_locales` — product is hidden in locales not listed (empty/absent = visible everywhere)
- Dates are ISO 8601 strings; IDs are UUID strings
- `thumbnail_url` lives in `ProductCardItem` (catalog list), not in `Product` — use `images[]` for full product
- When schema changes, regenerate with `pnpm orval` — never edit generated files in `src/api/stetsom/`
