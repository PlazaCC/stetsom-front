## Stack

## Path aliases

`@/*` maps to `src/` (confirmed in `tsconfig.json`: `"@/*": ["./src/*"]`). Write `@/lib/utils`, `@/components/ui/button`, etc. Do **not** write `@/src/...` — that resolves to `src/src/...`.

## Generated files

`types/routes.d.ts`, `types/cache-life.d.ts`, and `types/validator.ts` are auto-generated by Next.js. Do not edit them manually.

## Page and layout props

`params` and `searchParams` are `Promise` types in this version. Always `await` them:

```tsx
export default async function Page(props: PageProps<'/blog/[slug]'>) {
  const { slug } = await props.params
}
```

Use the global `PageProps<Route>` and `LayoutProps<Route>` types (declared in `types/routes.d.ts`) for page and layout component props.
